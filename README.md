# Sistema Bancario -- Backend

## Descripción

Este proyecto consiste en el desarrollo del backend de una aplicación
bancaria. El sistema permitirá gestionar clientes, cuentas, tarjetas y
operaciones financieras como consignaciones, retiros y transferencias.

La primera entrega se enfoca en:

-   Migración y diseño de la base de datos
-   Creación de APIs REST

------------------------------------------------------------------------

## Tecnologías utilizadas

-   Python\
-   FastAPI \
-   PostgreSQL \
-   Git y GitHub

------------------------------------------------------------------------

## Modelo de Base de Datos

El sistema está compuesto por las siguientes entidades:

-   Banco
-   Cliente
-   Usuario
-   Cuenta
-   Tarjeta
-   Operacion

------------------------------------------------------------------------

## Funcionalidades iniciales (APIs)

### Clientes

-   Crear cliente
-   Consultar clientes
-   Actualizar cliente
-   Eliminar cliente

### Cuentas

-   Crear cuenta
-   Consultar cuenta
-   Consultar saldo

### Operaciones

-   Consignar
-   Retirar
-   Transferir

------------------------------------------------------------------------

## Migraciones

Se implementará el sistema de migraciones para:

-   Crear las tablas iniciales
-   Definir claves primarias y foráneas
-   Garantizar integridad referencial

------------------------------------------------------------------------

## Objetivo de la Primera Entrega

-   Base de datos correctamente modelada.
-   Migraciones funcionales.
-   Endpoints CRUD básicos funcionando.
-   Conexión exitosa entre API y base de datos.
